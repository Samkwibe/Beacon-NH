Beacon NH — local API + MySQL (quick steps)
===========================================

1) Start MySQL (Docker Desktop must be running):

   npm run db:up

   First boot creates tables from server/schema.sql automatically.

2) Firebase Admin (same project as your web app VITE_FIREBASE_*):

   Firebase Console → Project settings → Service accounts → Generate new private key.

   Save the downloaded file as:

     server/firebase-admin.json

   Or copy server/firebase-admin.example.json → firebase-admin.json and replace
   every placeholder with values from the real JSON (do not commit firebase-admin.json).

3) Server env:

   server/.env should match docker-compose MySQL credentials (see server/.env.example).

4) Install & run API:

   npm run api:install
   npm run api:dev

   Health check: http://localhost:3001/api/health
   Expect mysql: true and firebaseAdmin: true after step 2.

5) Frontend:

   Root .env needs:

     VITE_API_URL=http://localhost:3001

   Then:

     npm run dev

Production: use a managed MySQL URL, set CORS_ORIGIN to your site origin, and set
FIREBASE_SERVICE_ACCOUNT_JSON or GOOGLE_APPLICATION_CREDENTIALS on the host (never commit keys).
